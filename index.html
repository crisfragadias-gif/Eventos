Como usar

  1. Confira os eventos de amanhã.
  2. Clique em "Adicionar ao Google Agenda" para criar o lembrete.
  3. Compartilhe o calendário com a equipe.
  4. Use "Copiar resumo" se quiser enviar no WhatsApp.

Passo a passo para usar o calendário compartilhado

  1. Exporte a semana pelo botão "Exportar semana para calendário".
  2. Abra o Google Agenda.
  3. Importe o arquivo .ics no calendário compartilhado do setor.
  4. Confira se os eventos apareceram corretamente.
  5. Os integrantes que tiverem acesso ao calendário receberão as notificações conforme as configurações do Google Agenda.

A exportação .ics não envia mensagens por WhatsApp. Ela serve para alimentar o calendário compartilhado.
